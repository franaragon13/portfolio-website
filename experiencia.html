Buscador de aceites esenciales y aromaterapia

Este proyecto se realizó para la asignatura de Desarrollo Web en Entorno Cliente y consiste en una web para una tienda online de aceites y aromaterapia. Para el desarrollo de este proyecto se usó JavaScript con React como framework.

El funcionamiento consiste en que al cargar la web no aparecerá ningún producto y al escribir en cualquiera de los campos de textos se realizará una búsqueda filtrando todos los productos almacenados en un JSON (dicha búsqueda podría ser complementaria por ambos campos).

Horario de un centro escolar

Este proyecto se realizó para la asignatura de Desarrollo Web en Entorno Servidor y consiste en una web para observar los horarios de todos los profesores de un centro escolar filtrando por aula. online. Para el desarrollo de este proyecto se usó JavaScript con PHP.

La web cuenta con un login el cual comprobará en la base de datos el usuario y la contraseña (en caso de ser correcto podrá acceder y en el contrario saltará el control de errores). Una vez logueado, si el usuario es el profesor1 (administrador), podrá añadir a cualquier profesor a la lista de ausentes de un determinado día y, automáticamente, se refescará una tabla situada justo debajo añadiendo en cada tramo horario de ese determinado día todos los profesores de guardia. Si por el contrario el usuario es otro profesor podrá ver su horario de toda la semana.
