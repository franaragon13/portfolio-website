Etapa académica

I.E.S Mar de Alborán

Realicé el ciclo formativo de grado superior de Desarrollo de Aplicaciones Web, finalizándolo en el año 2.021 con una nota media de X.

En mi etapa estudiando este grado adquirí conocimientos de lenguajes frontend como JavaScript, HTML5 o CSS3, lenguajes backend como PHP (programando tambien con el framework Slim3) o Java y algunos conocimientos básicos sobre redes. También usábamos herramientas para el diseño de prototipos de web como Figma y GIT como controlador de versiones (trabajando con varias ramas y colaborando en repositorios de compañeros de clase). A destacar también el uso de JQuery en algunas de las asignaturas mediante el cual creábamos animaciones o accedíamos a nodos del documento.
